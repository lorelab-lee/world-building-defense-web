📱 앱으로 설치해주세요

모바일 브라우저로는 주소창이 게임을 가려요.
앱으로 설치하면 전체화면으로 쾌적하게 플레이됩니다.

📲 바로 설치

버튼을 탭하면 설치 다이얼로그가 뜹니다

📲 Samsung Internet 설치 방법

방법 1 (빠름):

① 하단 메뉴(≡) 탭

② "추가" 아이콘 탭 (첫 줄)

③ "홈 화면에 추가" 선택

방법 2 (위가 안 보이면):

① 메뉴 → "데스크톱 사이트" 켜기

② 주소창의 ⬇ 아이콘 탭 → "설치"

📲 Chrome 설치 방법

① 우상단 메뉴(⋮) 탭

② "앱 설치" 또는 "홈 화면에 추가" 선택

③ 설치 확인

📲 앱 설치

브라우저 메뉴에서 "앱 설치" 또는 "홈 화면에 추가"를 찾아주세요.

최적 경험을 위해 Chrome 또는 Samsung Internet 사용을 권장합니다.

🍎 설치 방법 (Safari)

① 아래 공유 버튼을 눌러주세요 👇

② 메뉴에서 "홈 화면에 추가" 선택

③ 홈 화면의 아이콘으로 실행

⚠️ Safari로 열어주세요

iOS에서는 Safari에서만 앱 설치가 가능합니다 (Apple 정책).

아래 주소를 복사한 뒤 Safari에서 열어주세요:

수동 설치

브라우저 메뉴에서 "앱 설치" 또는 "홈 화면에 추가"를 찾아주세요.

설치 후

홈 화면의 월드 빌딩 디펜스 아이콘으로 실행하면 자동으로 가로 전체화면 진입합니다.
